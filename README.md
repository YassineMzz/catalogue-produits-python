# Automatisation de catalogues produits – Python

## Description
Script Python permettant de nettoyer et transformer un catalogue produits
au format CSV afin de le rendre exploitable pour une intégration e-commerce.

## Fonctionnalités
- Lecture de fichiers CSV
- Nettoyage des données produits
- Calcul du prix TTC
- Export d’un fichier prêt à l’intégration

## Technologies
- Python 3
- CSV

## Contexte
Projet réalisé dans le cadre d’une reconversion professionnelle
vers le développement Python et web.
